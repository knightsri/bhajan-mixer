================================================================================
  KEY PRINCIPLES
================================================================================

1. Dockerfile and wrapper scripts are ALREADY DONE
   - Focus on creating bhajan-mixer.py and requirements.txt

2. ALWAYS test through wrapper scripts
   - ./bhajan-mixer.sh --album "Test" <sources>
   - Never run docker/python commands directly

3. Build in phases
   - Phase 1: Core audio (YouTube + rotation)
   - Phase 2: Metadata (ID3 tags, flags)
   - Phase 3: Video support
   - Phase 4: Polish

4. Circular rotation algorithm is the core
   - Shorter sources wrap around
   - Total tracks = longest source length
